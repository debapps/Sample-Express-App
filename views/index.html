AWS (Amazon Web Service) is the cloud platform provided by Amazon. AWS provides servers and services on demand basis those are highly scalable.

AWS Global Infrastructure

  • AWS Regions
  • AWS Availability Zones
  • AWS Data Centers
  • AWS Edge Locations / Point of Presence

AWS Main Services

  • IAM - Access and Role Management
  • EC2 - Virtual Server
  • EBS, EFS - Elastic Block Storage & Network File system
  • S3 - Bulk Storage
  • Load Balancers (ALB, NLB, GLB) - Load Balance Service
  • Auto Scaling Groups (ASG) - Scaling Resources
  • RDS - Relational Database
  • ElastiCache - Cache Service
  • Amazon Aurora - Relation Database (Amazon Specific)
  • Route 53 - DNS register & DNS Services
  • VPC - Virtual Private Cloud
  • Container Services (ECS, EKS, ECR) - Docker Container and Registry
  • Elastic Beanstalk - Code Deployment Service
  • CICD - CodeCommit, CodeBuild, CodeDeploy, CodePipeline, Codestar, CodeGuru, CodeArtifact
  • DynamoDB - NoSQL Database Service
  • AWS Lambda - Serverless Function
